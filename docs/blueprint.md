# **App Name**: RQ Portfolio

## Core Features:

- Hero Section: Displays the name, impactful tagline, and navigation buttons.
- About Me Section: Showcases a professional photo, short bio, and objectives.
- Academic Background: Lists educational history with relevant dates.
- Professional Experience: Details previous work experience, highlighting key accomplishments.
- Technical Skills Showcase: Showcases languages, databases, tools, and hardware skills, presenting them with consideration given to experience level, with more sophisticated/recent technologies receiving greater visual emphasis using an LLM "tool" for ranking and categorization.
- Courses and Certifications: Lists completed courses and certifications, including issuing institutions.
- Contact Section: Provides contact information via email, phone, and location in an stylized section.

## Style Guidelines:

- Background color: Dark grey (#222222) to provide a modern, tech-focused feel.
- Primary color: Deep red (#BB2222) to add contrast and energy.
- Accent color: White (#FFFFFF) to provide readability and a clean aesthetic.
- Font: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text, 'Inter' (sans-serif) for longer body text. 
- Minimalist icons for each section to enhance user experience.
- Clean, responsive layout ensuring seamless access across devices.
- Subtle transitions and animations for a modern and interactive experience.
